如何搭建个人Hugo博客

Feb 21, 2020 · 1 min read

瑞翼工坊假期任务如何搭建个人Hugo博客

Windows下的安装

开始

  • 下载Hugo

    1. 首先到Hugo Releases查看Hugo版本

    2. 对应系统来选择Hugo版本(32位和64位)

      因为在之后配置中有可能会遇到一些需要额外拓展的问题,可以直接选择带有拓展的版本

      另外有使用git配合go的安装,但是考虑到大家并不是每个人都可以科学上网,所以在此不再重点描述

  • 配置Hugo

    1. 下载之后得到压缩包

    2. 解压至任意盘符

    3. 将解压之后的文件夹添加进系统变量中

      1. 右键点击此电脑点击属性,打开属性界面,点击高级系统设置

      2. 打开系统属性以后,点击环境变量按钮

      3. 打开环境变量以后,可以选择用户变量和系统变量,如果只是一个人用电脑的话,两个都可以,此处演示的是使用系统变量,找到Path

      4. 点击Path以后得到一个编辑环境变量界面,点击新建,将刚刚解压出来的文件夹的路径复制粘贴进去 如图最下面的那条是我的Hugo,版本跟之前演示的版本低,但没有影响

      5. 打开cmd,输入hugo version,如果显示Hugo版本的话,到此Hugo配置成功

  • 开始创建个人博客

    1. 任意一个位置下,此处我演示的为E盘

    2. 创建博客文件夹

      使用cmd来到选择的位置下,输入hugo new site blog(自定义博客文件夹的名字) 此处已生成myblog文件夹 打开后是这样的 比较关键的几个文件有

      • config.toml 配置文件
      • content 内容文件夹
      • data 数据文件夹
      • layouts 布局文件夹
      • static 静态文件夹
      • themes 主题文件夹

    3. 选择一个主题

      1. 在Hugo主题网页中选择自己喜欢的主题

      2. 这里我选择的是soho主题

      3. 选择的主题的页面中稍微往下拉一些,会看到安装教程 此处是进入博客文件夹根目录的themes文件夹中使用git安装, git之前的课上已经有具体教学, 此处不再过多赘述

        不过也有例外, 例如此处是在博客文件夹的根目录进行git安装的主题—m10c,这个主题好像有些问题, 尝试解决过, 但是失败了, 解决的可以教一下我

      4. 在cmd中打开博客根目录文件夹,进入themes文件夹,按照网页上的git指令进行安装,如图所示,soho主题已经被下载下来

      5. 返回博客根目录文件夹,切记不要停留在themes文件夹中

        启动Hugo服务 输入 hugo server -t soho –buildDrafts
        当cmd显示如下的界面时,启动服务成功

        此时打开浏览器,输入url为localhost:1313, 显示如下页面时, 个人博客已经创建成功

  • 添加修改博客内容

    1. 修改博客主页内容
      由于每个人选择的主题都不同,所以我没法能过说清楚全部主题的配置,甚至就比如我之前所说的m10c,是有问题的,并且我无法解决

      但是如果一个主题能用的话,大体上的修改配置是差不多的,首先进入主题文件夹,打开所选择的主题的文件夹,例如此处我打开的soho文件夹
      点击exampleSite文件夹,这个就是开发这个主题的作者留给我们的配置文件

      打开这个文件夹之后,看到这个config.toml文件,这个就是开发者留给我们的主题配置文件

      打开后是这样的, 每个主题的配置文件不一样,但是英语都是相同的,实在不行的话一个一个删或者改来试试起什么作用,在毫无头绪的情况下,也是一种调试思维

    2. 添加博客内容 还记得之前的项目根目录文件夹中的content文件夹么

      我们现在打开它,里面是空的

      此时使用cmd进入到当前目录中 输入 hugo new post/blog.md 这条指令已经在content文件夹中生成了一个post文件夹, 并且创建了一个blog.md文件

      这个文件后缀为md, 意思是markdown格式的文件,打开这个文件看一下

      之后再刷新我们之前的个人博客,就会发现一个名为Blog的题目已经显示出来了, 日期和我们之前的那个blog.md一样的, 说明这篇文章是我们之前创建的blog.md

      我们可以修改一下blog.md, 由于这是markdown格式的文件, 需要了解一些markdown, 但是基础使用很简单, 稍微看一看就能够掌握用法,这里添加三个标题

      我们再回头看看网页上, 三个标题已经成功显示在网页上, 到此, 博客的搭建和修改添加已经完成
